API 概览

REST + WebSocket 接口，Base URL: http://localhost:8080

快速开始

  1. 在「API Key」面板创建密钥，请求头携带 Authorization: Bearer nx_xxx
  2. 选择计费方式：包月套餐（固定配额）或按次计费（余额扣减）
  3. 调用 POST /api/v1/task 提交任务，POST /api/v1/execute 启动执行
curl -X POST http://localhost:8080/api/v1/task \
  -H "Authorization: Bearer nx_your_api_key" \
  -H "Content-Type: application/json" \
  -d '{"instruction": "执行无人救援任务..."}'

模型与定价

支持包月套餐与按次计费两种模式，超出套餐配额自动按次扣费
